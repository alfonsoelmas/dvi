# Frogger
Para esta práctica se han implementado las siguientes dinámicas:
- Escenario de fondo
- Movimiento de la rana
- Coches
- Colisiones
- Troncos
- Tortugas
- Agua
- Animacion Muerte
- Menus y condiciones de finalizacion con aumento de dificultad por cada WIN
- Generadores de coches y troncos
- Animacion de la rana
- Animacion de las tortugas (Sin desaparecer)
- Vidas (Sin mostrar, solo a nivel logico)
